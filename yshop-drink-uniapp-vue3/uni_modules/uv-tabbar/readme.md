## Tabbar 底部导航栏

> **组件名：uv-tabbar**

此组件提供了自定义`tabbar`的能力。

一般固定在底部，可以满足某些需要鉴权跳转等场景。一旦自定义底部导航栏后，为了达到不闪烁的效果，建议在`tabbar`页面进行组件切换。
